# Rubic

> 响应式小程序框架

[官方文档](https://rubic.vercel.app)

# 特性

- 响应性：基于 @vue/reactivity 的响应式能力
- 组合式：与 vue3 一致的 Composition API
- 类型化：灵活的 API 和完整的 TypeScript 类型。
- 轻量级：仅 30KB 的体积，无编译依赖，现有小程序可轻松接入。
