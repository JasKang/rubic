# Rubic

> 基于 Vue3 的小程序开发框架

# 特性

- 响应性：基于 @vue/reactivity 的响应式能力
- 组合式：与 vue3 一致的 Composition API
- 类型化：丰富的 Typescript 类型提示
- 轻量级：不依赖编译，原生小程序可直接接入
